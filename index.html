⚠️
All 17 parties MUST be listed on your sheet in the exact order and abbreviation shown below.
Write the party code exactly as shown — brackets required e.g. [A] not just A. The brackets help the AI identify party codes accurately.
Write 0 for any party with no votes — do NOT leave the row blank.
Numbers must be clear and readable for the AI to extract correctly.

Demo System — Please Read
This is a public demonstration of the NaijaVote collation platform, not a live election. Because this demo runs without a dedicated database, data entered by multiple users at the same time may overwrite each other — this is a known limitation of the current demo mode and does not reflect how the production system will behave. In the full version, every submission is written atomically to Firebase Firestore with conflict protection.
⏰ All data resets automatically at midnight (Nigerian time) every day so fresh testers get a clean slate.
🙏 Once you have tested the system, please allow others to do the same.
