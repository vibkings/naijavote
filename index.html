Demo System — Please Read
This is a public demonstration of the NaijaVote collation platform, not a live election. Because this demo runs without a dedicated database, data entered by multiple users at the same time may overwrite each other — this is a known limitation of the current demo mode and does not reflect how the production system will behave. In the full version, every submission is written atomically to Firebase Firestore with conflict protection.
Your participation helps test the interface. Results shown here are not real election data.
